Profile

I am a mathematician, programmer and data scientist passionate about learning. At work I build and implement time-series regression models for the retail fuel industry, whilst in my spare time I like to play games and think about ways to create AI that could beat me every time. I'm particularly interested in stochastic, imperfect information multiplayer games which demand complex, adaptable strategies and have most recently been reading about an extension to Monte Carlo Tree Search which tackles the vast state space of these games by using sampling. Excitingly the framework also allows for analysis of competitor behaviour and uses this to inform priors on distributions of the unknown elements before sampling.
